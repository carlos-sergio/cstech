📚 Como publicar meu projeto gratuitamente?

Opções rápidas e gratuitas:

GitHub Pages (Recomendado):
  1. Crie conta em GitHub
  2. Faça upload dos seus arquivos HTML/CSS/JS
  3. Ative GitHub Pages nas configurações do repositório
  4. URL ficará: username.github.io/nome-repositorio
Netlify Drop (Mais fácil):
  1. Acesse Netlify Drop
  2. Arraste e solte a pasta com seus arquivos
  3. Pronto! Site publicado instantaneamente
CodePen (Para projetos menores):
  1. Crie conta no CodePen
  2. Cole seu HTML, CSS e JavaScript
  3. Salve como "Public" e compartilhe o link
